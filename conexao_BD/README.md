# Conexão com o Banco de dados SQL Server na AWS

A conexão foi realizada atraves do Python em um Jupyter.

# Libs utilizadas
* Pyodbc: Comando pip install pyodbc
* Sqlalchemy: Comando pip install sqlalchemy
* Boto3: Comando pip install boto3
* Pandas: Comando pip install pandas

# Detalhes
Neste repositorio voce verá:
* Como realizar a conexão ao Banco de dados SQL Server
* Realização de uma consulta simples para vericação da conexão
