# Conexão ao S3 da AWS

Neste repositorio veremos como realizar uma conexão ao seu S3 com Python em um Jupyter.

# Libs utilizadas
* boto3: Comando pip install boto3
* Pandas: Comanda pip install pandas
* Os

# Detalhes
Neste repositorio voce verá:

* Como realizar conexao com o S3
* Criação do Client
* Criação do Resource
* Criação de Bucket
* Exportação dos dados locais para o novo Bucket
